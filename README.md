# Sentiment Analysis + EDA Project

## Description
This project is based on Exploratory Data Analysis (EDA) and Sentiment Analysis using Excel.

## Work Done
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sentiment Classification (Positive, Negative, Neutral)
- Data Visualization using Pie Chart

## Tools Used
- Microsoft Excel

## Outcome
Clear insights into sentiment distribution and data patterns.
